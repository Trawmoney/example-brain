### Tweet
Tip for thread writers:

3 tweets show in the feed:

• The top tweet
• The 2nd-to-last tweet
• The last tweet

The rest of it is condensed. So spend more time on these 3.

• Top tweet = hook
• 2nd tweet = TL;DR
• Last tweet = CTA
